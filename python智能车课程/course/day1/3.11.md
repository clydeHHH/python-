##### 切片

```python
name = 'buaaer'
print(name[1:4])#uaa
print(name[1:])#uaaer
print(name[:4])#buaa

```

#####  time库

```python
import time
time.time()
time.ctime()#显示北京（当前时区）时间
time.gmtime()#以time库中规定的格式显示格林尼治时间
#格式化控制
t = time.gmtime()
time.strftime("%Y-%m-%d %H:%M:%S",t)#各%为gmtime规定的控制符

#测量时间
time.perf_counter()
time.sleep(t)#t  推迟执行的秒数
```
